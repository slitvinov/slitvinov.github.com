bond_style stretching command

Syntax: 

bond_style stretching

Examples: 

bond_style stretching
bond_coeff 10.0 0.0 0.0 10 20 -60 60 -60 60

Description:

The stretching bond applays a constant force Fstr to the first atom of the bond and -Fstr to the second atom of the bond. Force acts if the first atom is in the region:

X0<x<X1, Y0<y<Y1, Z0<z<Z1 (1)

For the linear chain if all atoms are in the region application of the bond style is equaivalent to applying force Fstr to the head of the chain and force -Fstr to the tail of the chain.

In the illustration bellow stretching with force Fstr=(10, 10, 0) is applied in the region 10<x<20. The simulation domain is [0:30][0:20][0:20]. animation

The following nine coefficients must be defined for each bond type via the bond_coeff command as in the example above, or in the data file or restart files read by the read_data or read_restart commands:

  • FstrX FstrY FstrZ (force units)
  • X0 X1 Y0 Y1 Z0 Z1 (distance units)

Restrictions:

This bond style can only be used if LAMMPS was built with the "USER-STRECHING" package (which it is by default). See the Making LAMMPS section for more info on packages.

Related commands:

bond_coeff, delete_bonds

Default: none
